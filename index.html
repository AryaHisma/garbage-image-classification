Deskripsi Proyek

Garbage Image Classification adalah aplikasi berbasis Artificial Intelligence (AI) yang dirancang untuk mengklasifikasikan berbagai jenis sampah melalui gambar. Aplikasi ini merupakan versi pengembangan dari proyek sebelumnya yang hanya mampu mengenali tiga kategori sampah, dan kini telah ditingkatkan menjadi mampu mengklasifikasikan hingga sepuluh kategori, yaitu: battery, biological, cardboard, clothes, glass, metal, paper, plastic, shoes, dan trash.

Cara penggunaan: Pengguna cukup mengunggah foto sampah melalui browser, kemudian sistem akan secara otomatis mengklasifikasikan jenis sampah berdasarkan model AI yang telah dilatih sebelumnya. Versi terbaru ini dilengkapi dengan tampilan antarmuka baru serta fitur feedback untuk mengevaluasi kinerja model saat digunakan dalam kondisi dunia nyata.

Model dilatih menggunakan FastAI dan di-deploy ke web menggunakan ONNX Runtime Web. Pendekatan ini memungkinkan aplikasi dijalankan langsung di browser tanpa memerlukan server tambahan maupun instalasi khusus. Berbeda dari versi sebelumnya yang menggunakan model MobileNetV3-Small (ringan dan cocok untuk perangkat seluler), versi ini memanfaatkan ResNet18 yang lebih besar namun tetap efisien dijalankan di perangkat seluler berkat format ONNX.

Dataset

Proyek ini menggunakan dataset publik dari Kaggle: Garbage Classification V2 .

Dataset ini digunakan sebagai dasar pelatihan model agar mampu membedakan karakteristik visual antar kategori sampah dengan tingkat akurasi yang baik.

Tools

  • FastAI
  • ONNX Runtime Web
  • HTML, CSS, dan JavaScript

Project Description

Garbage Image Classification is an Artificial Intelligence (AI)-based application designed to classify different types of waste from images. This project is an upgraded version of the previous model, which could only recognize three waste categories. The new version has been enhanced to classify up to ten categories: battery, biological, cardboard, clothes, glass, metal, paper, plastic, shoes, and trash.

How it works: Users can simply upload a waste image through the browser, and the system will automatically classify the type of waste based on a pre-trained AI model. This version also includes a redesigned interface and a feedback feature to evaluate model performance in real-world scenarios.

The model was trained using FastAI and deployed to the web via ONNX Runtime Web. This approach allows the application to run entirely in the browser ‚Äî no server or installation required. Unlike the previous version that used MobileNetV3-Small (lightweight and suitable for mobile devices), this version uses ResNet18, a larger model that still performs efficiently on mobile devices thanks to the ONNX format.

Dataset

The project uses a public dataset from Kaggle: Garbage Classification V2 .

This dataset serves as the foundation for model training, enabling the system to distinguish visual differences between categories with high accuracy.
